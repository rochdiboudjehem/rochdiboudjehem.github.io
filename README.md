# Rochdi Boudjehem's CV Website

This repository contains the source code for a personal CV website built with [Vite](https://vitejs.dev/) and [React](https://react.dev/). The site showcases the professional experience, skills, and projects of Rochdi Boudjehem.

## Features

- Fast development with Vite
- Modern React components
- Responsive and clean design
- Easy to update and maintain

## Getting Started

1. **Install dependencies:**
    ```bash
    npm install
    ```

2. **Start the development server:**
    ```bash
    npm run dev
    ```

3. **Build for production:**
    ```bash
    npm run build
    ```
